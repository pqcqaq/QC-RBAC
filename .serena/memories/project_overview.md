QC-RBAC 是一个 pnpm monorepo RBAC 基础工程，包含 `apps/backend`、`apps/web-frontend`、`apps/app-frontend`、`apps/oauth-test-provider`、`apps/oauth-test-application`、`apps/docs` 和 `packages/api-common`。后端使用 Express + TypeScript + Prisma + PostgreSQL + Redis + JWT + Socket.io；Web 管理端使用 Vue 3 + TypeScript + Element Plus + Pinia + Vue Router + Vite；Uni 移动端基于 unibest 结构，但 UI 已改为项目自定义组件、自定义 header、自定义 tabbar、自定义安全区布局；`packages/api-common` 统一权限常量、客户端枚举、请求适配器、API 工厂和下载抽象；`apps/docs` 是当前项目的开发者文档与实现导航。

项目目标是提供一套可扩展的认证、授权、多端接入基础工程，当前核心能力覆盖：RBAC、客户端校验、策略化认证、用户偏好持久化、OAuth/OIDC Provider 与第三方登录客户端、文件/附件管理、Excel 导出、删除引用检查、Web 控制台和 Uni 移动端接入。
